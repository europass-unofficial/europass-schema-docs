    3.14.1 EmploymentHistory Element Description¶

    <img>A diagram showing "EmploymentHistoryType" with an "Attributes" group containing "@ validFrom", "@ validTo", and "hr:EffectiveDateAttributeGroup". It also shows "EmploymentHistory" with a description "Employment History records. Details of a particular tenure of employment or work with an organization." and "EmployerHistory" with a description "Details of a particular tenure of employment or work with an organization."     Element Description Cardinality Rule Examples
    EmploymentHistory The prior and current details of a person's employment, work, or relevant experience as they would be reported in a professional profile, CV, resume, employment application, or similar document 0..1 N/A N/A

    Sub-elements
